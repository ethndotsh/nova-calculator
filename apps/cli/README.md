# @novacalc/cli

Complete basic (and not so basic) calculations from the command line.

## Installation

```bash
npm i -g @novacalc/cli
```

## Usage

```bash
ncalc "<expression>"
```

## Examples

```bash
ncalc "1+1"
2
```

```bash
ncalc "4*8/2+10"
26
```
```bash
ncalc "sin(45)"
```
